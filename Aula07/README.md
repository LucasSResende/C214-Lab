<h1 align="center">Aula-7 Teste de unidade</h1>

>Exercicio destinado aos testes de validade das funções que adicionam os filmes em um banco de dados de catálogo de mídia.

## <h1 align="center"> Requisiçoes</h1>
- /movie
- Validações: - validation.test.js - determina se os filmes estão com todas as suas características inseridas;


## <h1 align="center"> Execução</h1>

Depois de clonar o projeto na máquina, utilizar o comando:
- npm install --save jest

Caso tenha algum problema, certifique-se da porta utilizada e, se mesmo assim estiver com problemas, dê um ctrl+C no termial utilizado e repita o procedimento, verificando, também, user e senha utilizados no MongoDB.

## Ferramentas utilizadas:
- VS Code;
