<h1 align="center">Aula-6 Implementação Back-End</h1>

>Exercicio destinado para implementação de um Back-End utilizando javascript e node.js com o tema de uma biblioteca de filmes

## <h1 align="center"> Requisiçoes</h1>
- /movie
- POST (/create) - Cadastrar um novo filme no catalogo
- GET (/list) - Listar todos os filmes do catalogo

## <h1 align="center"> Execução das API's REST</h1>

Depois de clonar o projeto na máquina, utilizar os comandos:
- npm install
- npm start

Caso tenha algum problema, certifique-se da porta utilizada e, se mesmo assim estiver com problemas, dê um ctrl+C no termial utilizado e repita o procedimento, verificando, também, user e senha utilizados no MongoDB.

## Ferramentas utilizadas:
- MongoBD;
- VS Code;
- Postman (para simulação da interface gráfica em funcionamento).
